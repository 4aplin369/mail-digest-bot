# Mail Digest Bot

Telegram-бот для ручного обзора Яндекс.Почты без LLM и без платных API.

Бот подключается к Яндекс.Почте по IMAP, читает письма за последние `LOOKBACK_HOURS` и присылает в Telegram полный список найденных писем по команде `/digest`. Сверху он добавляет сводку по категориям: важное, оплаты/счета/квитанции, скидки/промо, вложения и остальное.

Каждое письмо приходит отдельной карточкой с кнопками:

- `Пометить прочитанным` - ставит IMAP-флаг `\Seen`.
- `В корзину` - переносит письмо в папку корзины через IMAP `MOVE`.

Сам просмотр дайджеста не помечает письма прочитанными: бот читает их через `BODY.PEEK[]`.

## Команды в Telegram

- `/start` или `/help` - показать справку.
- `/digest` - показать сводку и карточки писем с кнопками.
- `/status` - показать текущие настройки: папку, период поиска, лимит писем и источник ключевых слов.

Бот отвечает только чату из `TELEGRAM_CHAT_ID`. Сообщения из других чатов отклоняются.

## Как бот раскладывает письма

По умолчанию ищутся слова и фрагменты вроде:

- `оплат`, `счет`, `счёт`, `квитанц`, `штраф`, `налог`
- `банк`, `карта`, `подписк`, `списан`, `истека`
- `invoice`, `payment`, `bill`, `receipt`, `subscription`, `deadline`
- `договор`, `документ`
- `Sektaschool`, `Sektoschool`, `промокод`, `промо-код`

Промо-письма вроде `скидка`, `акция`, `sale`, `promo` больше не пропускаются: они показываются в общем списке и попадают в категорию `Скидки / промо / промокоды`. Письма без специальных признаков тоже показываются и получают метку `обычное`.

## Локальный запуск

1. Создай Telegram-бота через BotFather и получи `TELEGRAM_BOT_TOKEN`.
2. Узнай свой `TELEGRAM_CHAT_ID`, например через `https://api.telegram.org/bot<TOKEN>/getUpdates` после сообщения боту.
3. В Яндексе создай пароль приложения для почты/IMAP.
4. Скопируй `.env.example` в `.env` и заполни значения.
5. Запусти бота:

```powershell
python mail_digest_bot.py
```

Или в фоне на Windows:

```powershell
.\start_bot.ps1
```

Остановить фонового бота:

```powershell
.\stop_bot.ps1
```

После запуска напиши боту в Telegram:

```text
/digest
```

## Одноразовый запуск

Если нужно просто собрать дайджест один раз и отправить его в Telegram:

```powershell
python mail_digest_bot.py --once
```

Для проверки без отправки в Telegram:

```powershell
$env:DRY_RUN="true"
python mail_digest_bot.py --once
```

## Настройка слов

Можно полностью заменить список важных слов через переменную окружения:

```text
IMPORTANT_KEYWORDS=оплат,счет,налог,договор,invoice,payment
```

Если список пустой, используется встроенный набор.

## Настройка корзины

По умолчанию кнопка `В корзину` переносит письмо в IMAP-папку `Trash`.

Если у твоей Яндекс.Почты папка называется иначе, добавь в `.env`:

```text
YANDEX_TRASH_MAILBOX=Trash
```
